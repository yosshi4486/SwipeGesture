# SwipeGesture

## Usage
```
YourView()
    .onSwipe(leftAction: {
        // do something in left swipe
    }, rightAction: {
        // do something in right swipe
    })
```

## Installation
Please install this package(https://github.com/yosshi4486/SwipeGesture) via SwiftPM, using Xcode GUI package manager.
